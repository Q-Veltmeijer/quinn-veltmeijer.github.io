Multi-Threaded Boids

For an assignment I used Unity's Job's System and the Burst-Compiler, to simulate 4000+ interactive boids in Virtual Reality.

Using your VR Controllers you can influence the direction of the boids, the faster you move your controllers the more influence you will have on the boids.

World Sculptor

For an assignment I made a world sculptor, using Compute shaders and a Marching Cubes algorithm.

Unity Compute Shaders, and C# was used for this project. This was my first time working with Compute Shaders.

Unfortunately it is not perfect however, it is lacking certain features such as loading external obj files and does not work well with Amd Graphics-Cards, nor does it support using other primitives than the Sphere.

But it has the capability to load and save the current model and export it to an .obj, you can change the resolution of the model and it's size, and you can change the size of the tool, you can undo as well.

Though it has it's flaws, it was a very fun project to work on, and it was very interesting to learn about Compute Shaders and the Marching Cubes algorithm.

Space Witch Drive-through

With a group of other people I made a game in which you work inside of a Space With Drive-through.

In the game you are greeted by costumers and are tasked to create potions for them.

You can create the potions by going through the potion book, and inserting the ingredients into the fysical Couldron.

In Collaboration with an Interaction Designer, I worked on the Unity System that accepts the Arduino Input from the couldron and converts it to ingredients, and the remainder Unity-side Systems, such as the Costumers.

Which then gets used to determine whether or not the potion was correct.

GLFW Application with Skybox, lighting, First Person Movement and Multi-Threaded Plane Generation.

I am currently working on a C++ GLFW Project that utilizes Opengl and Glad to create an environment the player can traverse through.

The terrain heightmap is generated using octave perlin noise, and utilizes multi-threading. The work gets divided over batches which is based on the concurrency level. The fog and sun are created using GLSL Shaders.

You can load in .obj files into the application, though the .mlt file format does need to match for it to properly load Unfortunately.

This project is not yet finished.

The Unkilled Garden.

With a group of other students I worked on a game called "The Unkilled Garden".

In the game you traverse a cave, filled with sounds and life. The further your progress and the more valves your turn, the more the life seeps away.

I worked on the Gameplay aspects in this project, such as making sure the valves work and are interactable, making the climbing mechanic, the player controller.
